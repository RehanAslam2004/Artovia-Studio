# 🎨 Artovia Studio

> A modern full-stack marketplace for selling and delivering premium digital graphic designs.


## 📖 Overview

**Artovia Studio** is a full-stack web application built for creative entrepreneurs and design studios who want to sell and deliver digital graphic design products online. The platform supports a range of design products including wedding cards, invitations, posters, and creative templates.

Artovia Studio bridges the gap between designers and clients — providing a seamless storefront experience where customers can browse, purchase, and receive high-quality design assets instantly.

---

## ✨ Features

- 🛍️ **Digital Product Storefront** — Browse and purchase wedding cards, invitations, posters & templates
- 📦 **Instant Digital Delivery** — Customers receive their files immediately after purchase
- 🎨 **Design-First UI** — A visually rich interface that showcases creative products beautifully
- 👤 **User Accounts** — Customer registration, login, and order history
- 🔒 **Secure Transactions** — Protected order and delivery flow
- 📱 **Responsive Design** — Optimized for both desktop and mobile browsing
- 🗂️ **Product Management** — Admin-side tools to manage listings and deliveries

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/RehanAslam2004/Artovia-Studio.git

# 2. Navigate into the project directory
cd Artovia-Studio

# 3. Install dependencies
npm install

# 4. Start the development server
node server.js
```

### Access the App

```
http://localhost:3000
```
